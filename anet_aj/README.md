# Serpente — Visual identity project

Bachelor thesis project by **Aneta Bromovská** (VŠKK, 2026).

Complete visual identity for the Italian bistro Serpente in Roztoky.

## Live page

When this folder is placed at `homework/serpente/` in your `bromovskaaneta.github.io` repo, the project is live at:

**https://bromovskaaneta.github.io/homework/serpente/**

## Structure

```
serpente/
├── index.html             ← landing page with 5 tiles
├── about-me.html          ← who I am, how I worked
├── about-serpente.html    ← the bistro and its owners
├── values.html            ← five brand pillars
├── identity.html          ← logo, colour, type, packaging, print...
├── conclusion.html        ← what I learned
├── style.css              ← shared styles for all pages
├── images/                ← 16 images from the thesis
└── README.md              ← this file
```

## Sections

1. **About me & the process** — personal intro, how the project was built step by step
2. **What is Serpente** — the bistro, the owners (Nebojša Živanović & Karolína Chrdlová), the town
3. **Brand values** — five pillars: honesty, human touch, La Dolce Vita, local identity, visual nostalgia
4. **Visual identity** — the full system: logo, colour palette, typography, icons, packaging, print, exterior, online, guerrilla
5. **Conclusion** — three lessons from the project
